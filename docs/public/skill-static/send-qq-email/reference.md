# 能力展示

这个 skill 解决的是“让 Agent 帮我发一封 QQ 邮件”的问题。它可以先生成邮件快照，让你检查标题、正文和附件；确认后再发送。

![QQ 邮件发送展示](./assets/send-email-showcase.png)

## 它适合做什么

- 发送测试邮件。
- 发送日报、通知或整理好的材料。
- 把 PPT、论文分析结果或其他附件通过 QQ 邮箱发出去。
- 发送前先预览，避免内容写错或附件漏掉。

## 你会怎么用

你可以直接这样说：

```text
请把这段内容用 QQ 邮箱发给指定收件人，发送前先给我确认。
```

或者：

```text
请用 QQ 邮箱预演一封测试邮件，不要真实发送。
```

## 你会得到什么

Agent 会给你一份清楚的发送结果：

- 邮件是否已经发送。
- 如果只是预演，会明确告诉你没有真实发送。
- 邮件快照在哪里。
- 失败时，大概是什么原因。
