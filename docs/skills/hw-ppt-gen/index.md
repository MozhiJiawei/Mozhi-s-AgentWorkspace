# 华为风格 PPT 生成

本仓库提供 Huawei PPTX Generator skill：把论文、网页、Markdown、仓库分析或纯文本材料转成华为风格 PPTX，并通过计划、视觉锚点 manifest、PowerPoint COM 导出图和 QA 规则约束交付质量。

主仓文档按三段组织：

- [能力展示](reference.md)：展示 forward-test 最新运行的交付件、导出图和判题结果。
- [使用方式](usage.md)：说明输入、生成流程、产物目录和 QA 命令。
- [架构概览](architecture_design.md)：直接复用仓库维护态架构契约，说明 runtime、schema、composition、visual rendering、artifact 和 verification 的边界。
