# Skills

以下内容由各 skill 子仓的 `docs.manifest.yml` 集成生成。

## [ppt-deep-search](./ppt-deep-search/)

在生成 PPT 前进行观点对齐、故事线规划和证据审计。

## [huawei-pptx-generator](./hw-ppt-gen/)

基于材料生成华为风格 PPTX，并执行交付前 QA 和视觉检查。

## [gh-issue-comment-monitor](./gh-issue-comment-monitor/)

只读取相对本地 checkpoint 的新增 Issue 评论，避免重复加载完整讨论历史。

## [grobid-docling-pdf](./grobid_pdf_skill/)

将论文或技术 PDF 解析为包含正文结构和图表索引的 TEI/XML 包。

## [send-qq-email](./send-qq-email/)

通过 QQ 邮箱 SMTP 发送或 dry-run UTF-8 纯文本 / HTML 邮件。

## [[beta] generate-3plus1-diagrams](./architecture_4-1/)

分析目标仓库并生成可编辑的 draw.io 架构视图。
